Interactive fiction

A curated list

by Vincent Verhoeven (I actually don't know a thing about IF)

The following list is only inspired by my own experience of playing interactive fiction. It is not meant to be an exhaustive list. I will have missed important works, and my experience of certain works will have been colored by my cultural background at the time. That said, feel free to offer me feedback!

If you are stuck on any of the below games, and you want a gentle nudge in the right direction without huge spoilers, please ask me! Searching for walkthroughs on the Internet is risky!

  • 9:05

    Adam Cadre

    Official website

    Linear story. Very easy. Great introduction to the medium.

  • Photopia

    Adam Cadre

    Official website

    Super intense story. Mostly linear. You can't get stuck. Very light on puzzles. Does a couple of things that can only be done in the IF medium. Also a great introduction piece.

  • Lost Pig

    Admiral Jota

    Official website

    Really funny and well-implemented game. Some puzzles, but there are usually multiple solutions available.

  • Whom The Telling Changed

    Aaron A. Reed

    Official website

    A game about narrative. As player you have indirect control of the story, which makes for an interesting experience.

  • Coloratura

    Lynnea Glasser

    Official website

    You can only indirectly influence the game world. Very nice writing style.

  • Aisle

    Sam Barlow

    IFDB

    One-move game. But there is still a lot to discover. This is another game that pushes the IF medium to the boundaries.

  • All Things Devours

    half sick of shadows

    IFDB

    The whole game is a single puzzle, but it is wonderfully implemented and very satisfying to solve.

  • Spider and Web

    Andrew Plotkin

    Official website

    It is not the easiest or most forgiving game, so some IF experience is advised. But this game has The Best Moment in interactive narrative. A must play. Avoid spoilers.

  • Shade

    Andrew Plotkin

    Official website

    Interesting narrative.

  • Make It Good

    Jon Ingold

    IFDB

    Detective noir game. Many interactions are implemented masterfully, and there is more going on in the game than you would think at first, both on the mechanical and on the story level.

  • Vespers

    Jason Devlin

    IFDB

    Well-written story about a monastery that is slowly decaying into madness.

  • Slouching Towards Bedlam

    Star Foster and Daniel Ravipinto

    Official website

    Well-implemented and interesting steampunk story.

  • Savoir-Faire

    Emily Short

    IFDB

    Quite difficult and puzzly, but in a very intelligent way. Puzzles have multiple and sometimes even emergent solutions. This game is old-school in the sense that it's possible to lock yourself out of winning.

  • Rameses

    Stephen Bond

    IFDB

    The player character is a frustrated young adolescent, and this game plays with the IF medium to transfer this frustration onto the player.
